# Tekkaretest

## Overview

Tekkaretest is an application designed to display information about hospitals. It uses React with Vite for fast development and build processes. The app incorporates Shadcn React components and Tailwind CSS.

## Demo

You can see a live demo of the application [here](https://test-tekkare.netlify.app/).

## Technologies Used

- **Vite**:
- **React**:
- **Shadcn React**: A collection of UI components for React, including various Radix UI components.
- **Tailwind CSS**: A utility-first CSS framework for creating custom designs.

## Installation

To get started with Tekkaretest, clone the repository and install the dependencies:

```bash
git clone https://github.com/Matt-Fr/testTekkare
cd tekkaretest
npm install
```

## Scripts

The following npm scripts are available:

- `dev`: Start the development server with Vite.
- `build`: Build the project for production using TypeScript and Vite.
- `lint`: Run ESLint to check for code issues.
- `preview`: Preview the built project.
